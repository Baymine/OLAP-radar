**Title:** Enhancing OLAP Resilience at LinkedIn  
**Authors:** Praveen Chaganlal, Jia Guo, Vivek Vaidyanathan, Dino Occhialini, Sonam Mandal, Subbu Subramaniam, Siddharth Teotia, Tianqi Li, Xiaxuan Gao, Florence Zhang  
**Publication date:** 2026-03-07  

**Research problem:**  
How can real-time OLAP datastores preserve subsecond latency and high availability under operational disruptions such as noisy neighbors, node failures, rebalancing, and correlated infrastructure faults?

**Core method:**  
The paper presents a production resiliency framework for Apache Pinot built around four mechanisms: **Query Workload Isolation (QWI)** for fine-grained CPU/memory budgeting, **Impact-Free Rebalancing** for SLA-safe data movement, **Maintenance Zone Awareness** for replica placement across fault domains, and **Adaptive Server Selection** for routing queries away from overloaded or degraded nodes using live performance signals.
